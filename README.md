# Kola Test

Uma aplicação desenvolvida com Create React App.

## Scripts disponíveis

No diretório do projeto, execute o seguinte comando para instalar as dependências:

### `yarn install` ou `npm install`

Logo após, execute:

### `yarn start` ou `npm start`

O projeto será executado em modo de desenvolvimento e abrirá em seu navegador:
[http://localhost:3000](http://localhost:3000)
